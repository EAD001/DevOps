# DevOps Learning Portfolio

This repository documents my journey of learning DevOps and related technologies through consistent, hands-on practice. It serves as a structured record of projects, labs, and experiments across core DevOps domains.

---

## 🎯 Goals

- Learn real-world DevOps workflows used in modern cloud environments  
- Build strong hands-on experience with industry-standard tools and infrastructure  
- Develop the ability to design, deploy, and troubleshoot production-like systems  
- Document learning clearly and consistently to track progress and reinforce understanding  

---

## 📚 Learning Areas

### 🐧 Linux
- OverTheWire Bandit wargame
- Linux command-line fundamentals and daily practice
- File system, permissions, processes, and scripting basics

### 🐚 Bash Scripting
- Writing and automating tasks with Bash
- Bash Battle Arena challenges
- Working with loops, conditionals, and automation scripts

### 🧑‍💻 Git & Version Control
- Git fundamentals (commit, branch, merge, rebase)
- Collaborative workflows
- Managing repositories and version history effectively

### 🌐 Networking
- IP addressing, DNS, routing, and firewalls
- Troubleshooting connectivity issues
- Building simple cloud networks (DNS, EC2, HTTP services)

### 🐳 Containers & Docker
- Building Docker images using Dockerfiles
- Running and managing containers
- Working with Docker Hub (push/pull images)
- Multi-container application setups

### ☁️ AWS
- Designing custom VPC architectures (subnets, IGW, NAT Gateway)
- Deploying EC2 instances behind an Application Load Balancer
- Configuring security groups and network access controls

### 🏗️ Terraform
- Writing Infrastructure as Code (IaC)
- Deploying reproducible cloud environments
- Managing infrastructure across different environments

### 🔄 CI/CD
- Building CI/CD pipelines from scratch
- Automating testing, linting, and deployment workflows
- Understanding pipeline stages and automation best practices

### ☸️ Kubernetes
- Understanding container orchestration concepts
- Deploying workloads using Deployments, Services, Ingress, and ConfigMaps
- Running and managing local and cloud clusters
- Debugging and troubleshooting Kubernetes applications

## 🚀 About This Repository

Each section contains hands-on labs, notes, and small projects that demonstrate practical understanding rather than just theory. The focus is on building real-world DevOps skills through repetition and implementation.
